# Directus Masked Input Interface

Input text string by template

## Installation

In your Directus installation root

```bash
npm install dimitrov-adrian/directus-extension-masked-interface
```

Restart directus

## How to use it

# Masked text input

Handler

\+ ( **_ ) _** **\_ \_**

**\_** @ **\_** . \_\_

Or for example vehicles plate number like

P4096 HB

aa 9999 aa
